🏨 Hotel Booking System

A web-based Hotel Booking System developed using ASP.NET and C#.
This system allows users to view available rooms, book rooms online, and manage hotel booking information efficiently.

📌 Project Overview

The Hotel Booking System is designed to simplify the hotel reservation process.
Users can easily browse rooms, make reservations, and manage booking information through a simple and user-friendly interface.

This project demonstrates the implementation of web application development using ASP.NET architecture.

✨ Features

🛏️ View available hotel rooms

📅 Book rooms online

👤 Customer information management

💳 Online payment support

🧾 Booking details management

🔐 Admin management system

🚀 How to Run the Project
Step 1: Clone the repository

git clone https://github.com/your-username/HotelBookingSystem.git

Step 2: Open the project in VS Code

Open the folder in Visual Studio Code.

Step 3: Restore dependencies

Run the following command in the terminal:

dotnet restore

Step 4: Run the application

dotnet run

Step 5: Open in browser

After running the project, open the following link in your browser:

https://localhost:5001
or
http://localhost:5000

🎯 Purpose of the Project

The main goal of this project is to demonstrate how a hotel reservation system can be developed using ASP.NET web technology.
It helps students understand concepts such as web application architecture, booking management, and database interaction.
